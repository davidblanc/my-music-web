# Welcome to Music Web Site!

This is a [Remix](https://remix.run) app to show my music:
www.daviblanc.com

## development

run the dev server:

```shellscript
npm run dev
```

## deployment

first, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```

Now you'll need to pick a host to deploy it to.
